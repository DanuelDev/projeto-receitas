Instruções para rodar

Opção 1:
- Faça download do arquivo ZIP;
- Instale o XAMPP;
- Coloque a pasta do projeto dentro de `C:\xampp\htdocs\` (Windows);
- Inicie o Apache no painel de controle;
- No navegador, acesse `http://localhost/projeto-receitas/index.php`

Opção 2:
- Faça um fork do projeto e abra-o no Visual Studio;
- Instale a extensão "PHP Server" no Visual Studio Code;
- Clique no ícone azul do PHP localizado na parte superior direita do editor

Opção 3:
- Abra os arquivos com o Visual Studio Code, clique com o botão direito na pasta e clique em "Open In Integrated Terminal" (ou use o atalho Ctrl + ` );
- No terminal, execute:
  
```bash
php -S localhost:8000
```

- No navegador, acesse `http://localhost:8000/index.php`
